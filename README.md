# lightLabeler

个人使用的标注系统（前端部分）

## 声明

和后端代码一样，本项目目前只计划开放当前版本源码，以后的代码应该会闭源。

由于自己只能说略懂前端，所以开发的界面比较ugly，并且也不完善，许多都是试探性的尝试使用组件。

## 简介

该标注系统主要用于简单的标注任务，数据的标注信息会实时同步到数据库中，目前已经基本实现了文本分类任务（样本类别数量不大）。

## Project setup
```
npm install
```
